# Troubleshooting

- `gradlew.bat` is not found: run commands from `week5-lab/week5-day1-juc-pools-backpressure`.
- Java toolchain errors: install JDK 21 or newer.
- `studentCheck` fails on TODO methods: expected until exercises are implemented.
- Tests hang: look for missing `take`, `put`, `shutdown`, or `clear` behavior.
